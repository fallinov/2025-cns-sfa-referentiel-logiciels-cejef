# Accessibilité — ARIA & rôles

- Utiliser les rôles ARIA lorsque nécessaire (et seulement lorsque nécessaire)  
- Associer les éléments interactifs avec `aria-label`, `aria-describedby` si besoin  
- Assurer un focus visible pour tous les éléments interactifs  
- Gérer correctement les focus lors de l’ouverture/fermeture de modales  
- Utiliser `aria-live` pour annoncer les changements dynamiques importants (erreurs, succès)
