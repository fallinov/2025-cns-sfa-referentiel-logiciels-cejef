# Accessibilité & UX Mobile

- Touch targets suffisamment grandes (boutons, liens, icônes)  
- Espacements suffisants entre les éléments cliquables  
- Layouts responsives, évitant le scroll horizontal  
- Ergonomie mobile : menus clairs, formulaires simples, clavier adapté au type de champ  
- Prévoir des cibles d’au moins ~44px et des marges pour éviter les taps involontaires
