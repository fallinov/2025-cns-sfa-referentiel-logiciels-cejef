# Nuxt — Guide général

Bienvenue dans le guide Nuxt. Utilise ce fichier comme référence principale pour l’architecture d’un projet Nuxt 3.

## Principes clés
- pages/, layouts/, components/, composables/, server/ …
- SSR / SSG / auto imports / composables / routing

## Recommandations
- Toujours segmenter en composants réutilisables
- Éviter les fichiers « monstres » de pages
- Utiliser des composables pour la logique métier
