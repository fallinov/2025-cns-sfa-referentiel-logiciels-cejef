# Nuxt — Performance & optimisation

- Lazy-load des composants lourds avec `defineAsyncComponent`
- Minimiser le JS client inutile
- Utiliser SSR/SSG quand possible
- Pagination / lazy loading pour les listes volumineuses
- Optimiser les imports, éviter les librairies lourdes si Nuxt UI / Tailwind suffisent
- Favoriser le code-splitting (imports dynamiques) et limiter les polyfills inutiles
- Tester les performances (metrics Core Web Vitals) après chaque ajout majeur
