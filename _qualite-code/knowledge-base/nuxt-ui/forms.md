# Nuxt UI — Formulaires & validations

- Utiliser `<UForm>`, `<UInput>`, `<UButton>`, etc.  
- Associer correctement labels et champs pour l’accessibilité  
- Gérer les états d’erreur, `disabled`, `loading`, et le focus  
- Fournir des messages d’erreur clairs et utiles à l’utilisateur  
- Utiliser `aria-live` pour les messages d’erreur dynamiques et assurer un focus visible
- Valider côté client pour l’UX, toujours revérifier côté serveur pour la sécurité
