# Nuxt UI — Layouts & composants de mise en page

- Utiliser les composants de layout fournis quand ils existent  
- Ne pas redéfinir complètement le CSS de layout si Nuxt UI propose déjà un pattern adapté  
- Garder la mise en page cohérente d’un écran à l’autre  
- Assurer une hiérarchie claire (titres, sections) et un ordre de tabulation logique
