# Nuxt UI — Composants & usage

- Utiliser les composants standard (boutons, inputs, modales, etc.)  
- Respecter l’API (props, slots, events) de chaque composant  
- Ne pas reconstruire des composants déjà existants dans Nuxt UI  
- Personnaliser via props/classes/slots sans casser le comportement natif ni l’accessibilité
