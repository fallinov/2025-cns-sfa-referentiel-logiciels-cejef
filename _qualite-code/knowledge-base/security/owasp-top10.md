# Sécurité — OWASP Top 10 (résumé)

- Injection (SQL, NoSQL, etc.)  
- XSS / Cross-site Scripting  
- Problèmes d’authentification & gestion des sessions  
- Exposition de données sensibles  
- Mauvaise configuration de sécurité  
- Vulnérabilités liées aux composants  
- CSRF et autres vulnérabilités courantes  
- Logging et monitoring insuffisants
