# Tailwind CSS — Typographie & design system

- Utiliser les classes de typographie utilitaires (`text-sm`, `text-base`, `text-lg`, etc.)  
- Respecter la hiérarchie sémantique des titres (`h1`, `h2`, `h3`, …)  
- Veiller à la lisibilité (taille minimale, interlignage suffisant)  
- Garantir un contraste suffisant entre texte et fond  
- Contrôler la largeur de ligne pour la lisibilité (`max-w-prose`, `leading-relaxed`) et adapter la typo en mobile-first
