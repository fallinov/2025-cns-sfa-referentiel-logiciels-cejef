# Tailwind CSS — Espacements & layout

- Utiliser les utilitaires de spacing (`p-4`, `m-2`, `gap-4`, etc.)  
- Garder une cohérence d’espacement dans tout le projet (échelle de spacing définie)  
- Utiliser `gap` sur les layouts en grid/flex plutôt que multiplier les `margin` en tous sens  
- Ne pas mélanger des échelles d’espacement incohérentes dans le même écran  
- Appliquer la même échelle aux composants Nuxt UI (classes `class`/`ui`) pour rester aligné au design system
