Pourquoi classifier ?

La LGPD (Loi fédérale sur la protection des données) exige que les institutions publiques garantissent la sécurité des données de leurs usagers.

Pour le CEJEF, cela concerne directement les données des élèves utilisées dans les logiciels pédagogiques.

Risque légal

Amendes RGPD jusqu'à 250'000 CHF

Cloud Act

Les données sur serveurs US sont accessibles par la justice américaine

Devoir de protection

Données d'élèves mineurs sous notre responsabilité

LGPD — Loi fédérale sur la protection des données (Suisse) RGPD — Règlement général sur la protection des données (UE) Cloud Act — Loi américaine autorisant l'accès aux données sur infrastructure US

3 niveaux de classification

VALIDÉ

Usage autorisé

Données d'élèves autorisées. Création de comptes OK.

RESTREINT

Usage avec précautions

Enseignants uniquement. Pas de données d'élèves.

INTERDIT

Usage interdit

Ni enseignants ni élèves. Non conforme.

Règle clé : Le niveau global est déterminé par le critère le plus faible. Si l'hébergement est vert mais que la conformité RGPD est orange, le logiciel est orange.

Périmètre : Ce référentiel couvre uniquement les logiciels pédagogiques, pas les outils administratifs. Ils ne permettent pas de stocker des données sensibles (santé, notes officielles, etc.).
